sliding_window¶

data.helper_funcs.sliding_window(df, bins=datetime.timedelta(seconds=300), step=datetime.timedelta(seconds=120))¶

a sliding window to create smaller timeseries

Parameters

  • df (pandas dataframe) – input timeseries

  • bins (time window) – number of minutes (default= 5) to take for the window function

  • step (stride size) – decides the overlapp between the sliding window default is 2 minutes

Returns

chunk_list – a list containing the sliced dataframes

Return type

list of dataframes
