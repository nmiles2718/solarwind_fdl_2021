get_resampled¶

data.helper_funcs.get_resampled(df, binsize='3s')¶

resamples the pandas dataframe

Parameters

  • df (pandas dataframe) – timeseries data that will be resample

  • binsize (time in mins or seconds) – default is 3s, which means the timeseries is resampled to a frequency of 1 sample/3s

Returns

df_resampled – downsampled timeseries

Return type

pandas dataframe
