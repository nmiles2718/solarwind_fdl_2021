interpolate_orbit¶

data.prototyping_metrics.interpolate_orbit(date, orbit, kind='linear')¶

Function for interpolating an arbitrary time given the orbit of the spacecraft

Parameters

  • date (pandas.Timestamp) – Date when we want the postion of the instrument

  • orbit (pandas.DataFrame) – Dataframe holign information about the orbit of the instrument

  • kind (str) – Type of interpolation to use. Default ‘linear’

Returns

stats_data – dict containing the spacecraft position on the requested date

Return type

dict
